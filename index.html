Master Code

Your keys are stored encrypted inside the game. The master code decrypts them on this device — nothing is sent anywhere. Prefer manual entry? Use the fields below.

The Storyteller's Key

Powers the story. Create one at console.anthropic.com → API Keys. A few dollars of credit lasts a very long time (≈25¢ per hour of play).

The Painter's Key

Paints each scene live (~1–2s, ≈0.3¢ per image). Get one at fal.ai/dashboard/keys. Without it, scenes appear as elegant title cards — you can add it later in Settings.

🔒Your keys are stored only on this device and sent only to Anthropic / fal.ai. Never share this page's saved data with anyone.
